## 为什么需要这本指南？

Claude Code 是 Anthropic 官方推出的 CLI 编程助手，但官方文档仅覆盖基础功能。**实际使用中**，你需要了解：

- 如何配置 **Agents** 让 Claude 自动处理复杂任务
- 如何使用 **MCP** 连接 GitHub、数据库、Slack 等外部工具
- 如何编写 **Rules** 确保代码风格一致、安全可靠
- 如何利用 **Hooks** 实现自动化工作流（自动格式化、安全检查）
- 如何用 **Memory** 让 Claude 跨会话记住项目上下文
- 如何使用 **Everything Claude Code** 等生态工具大幅提升效率

本指南将带你系统掌握这些能力。
